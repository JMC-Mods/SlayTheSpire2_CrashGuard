# Changelog

All notable changes to this project are documented in this file.

Versioning rule: major.minor.patch. The major version is used for major feature improvements, the minor version is generally updated when a new Steam Workshop release is published, and the patch version is incremented after each code-related commit, starting from 0.

## [1.2.0] - 2026-7-9
### Added
- Added a guard for Ancient-related freezes or exceptions that may be caused by some mods, such as Hextech Runes, in certain situations.
- Added a guard for a native game crash that may occur in some environments when the game disables Sentry during modded startup.

## [1.1.12] - 2026-6-21
### Fixed
- Fixed compatibility issues introduced by the BaseLib 3.3.1 update.

## [1.1.10] - 2026-6-21
### Added
- Initial release.
